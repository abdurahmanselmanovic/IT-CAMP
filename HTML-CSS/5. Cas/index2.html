Kaskadni stilovi

Postoje 3 vrste primene kaskadnih stilova na HTML dokumentima

  1. Dirketna primena stilova kao atribut nekog taga koji zelimo da menjamo.
  2. Unutar head taga imamo style tag unutar kog vrsimo primenu CSS-a
  3. Eksterni CSS. Njega zapravo karaktrerise novi fajl (npr. style.css) koji ce biti povezan sa nasim html fajlom.
